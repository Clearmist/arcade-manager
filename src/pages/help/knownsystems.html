Known systems

Each system has strict requirements to load overlays, and they're all different (otherwise where's the fun?). This utility can only install overlays on know systems, with a fixed and predictable folder structure, because it has to define absolute paths (and replaces them on the fly while copying the overlays).

If you want to install on another system (like on your PC):

  • Create a config folder somewhere, let's say C:\Arcade\config
  • Run the utility, and select Recalbox.
  • Open the Install Overlays menu. It should now display text fields for rom and config.
  • Select the folder where your roms are located, and the folder you created, then run "Download and install".
  • Edit the paths in the downloaded cfg files:
    • Move the files located in overlays/fba_libretro/arcade-xxxx/ where you want (including the "common" folder).
    • Move the overlays/roms/romname.zip.cfg files where they need to be; it depends on the system: next to the roms? In a special folder? Check the doc, ask around.
    • In the romname.zip.cfg file, edit the paths to point to the config files
