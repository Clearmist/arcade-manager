Install and configure arcade systems

Installing and updating individual emulators is only available on Retropie.

Update Retropie and the emulators to the latest versions. When asked, choose "update from source": it will take a while (might take a few hours or even a few days), but it will get the latest version, which can include many more games, be much more stable, etc.

Libretro emulators (lr-mame2003, lr-mame2010, lr-fbneo) use the Libretro general configurations, so you shouldn't have troubles using them.

Go to the arcade documentation to check for the configuration paths for MAME4ALL, AdvanceMAME and PiFBA.
