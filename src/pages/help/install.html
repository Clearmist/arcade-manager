Use the setup script to update the emulators to the latest versions. When asked, choose "update from source": it will take a while (might take a few hours), but it will get the latest version, which can include many more games, be much more stable, etc.

Libretro emulators (lr-mame2003, lr-mame2010, lr-fbalpha) use the Libretro general configurations, so you shouldn't have troubles.
Check the MAME and FBA wiki pages to check for the configuration paths for MAME4ALL, AdvanceMAME and PiFBA.
