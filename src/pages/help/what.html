What to do with this tool

Feeling a bit lost with what this tool can be used for?
Installing overlays should be pretty straightforward. If it's not, please open an issue on the GitHub project.
For the rest, read on.

An example of typical usage

Let's say you want to play some arcade classics on your Raspberry Pi using Retropie:

  1. On the main menu, select Manage games list, then Download a pre-build CSV file. FBA is the preferred emulator by default, so click on FBA 2017.
  2. If you have a game pad with analogue sticks (like Xbox or Playstation), download set-classics-pad. Otherwise, download set-classics-stick.
  3. Go back to the main menu, and select Manage roms.
  4. If you have a full FBA romset somewhere, select Copy listed roms from a full romset. If you already have a large selection of working games on your pi, and want to trim it down, select Keep only the roms listed.
  5. Select the CSV file you previously downloaded, fill the folders as required (target \\retropie\roms\fba), then enjoy your great-but-smaller games selection.

FBA is great, but it's missing a few games and systems, like vector games (Asteroids...). If you don't want to miss anything:

  1. Go back to downloading games list, but choose MAME 2003 only as the system.
  2. Go back to managing roms, and do the same steps, using a MAME 2003 romset and target folder (\\retropie\roms\mame-libretro).
